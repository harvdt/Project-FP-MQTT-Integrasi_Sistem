## Project 1

### Setup Hardware

Berikut adalah hardware yang diperlukan

-   NodeMCU ESP8266
-   Sensor DHT22
-   Kabel Jumper
-   Breadboard

Berikut adalah konfigurasi hardware

-   DHT22 Pin **OUT** - ESP8266 Pin **D1** (GPIO5)
-   DHT22 pin **(+)** - ESP8266 pin **3V**
-   DHT22 pin **(-)** - ESP8266 pin **GND**
